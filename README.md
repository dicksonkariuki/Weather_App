# Weather_App
This repository contains a 2 tier weather app containerized in docker and deployed using docker compose

# The technologies to use are :
***React Frontend***

***Node.js + Express Backend***

***OpenWeatherMap API***

***Docker for containerization***

# Features

***Enter city name***

***Fetch current weather***

***Display***

***Temperature***

***Humidity***

***Wind Speed***

***Weather Description***

***Weather Icon***

***Responsive UI***

***Dockerized application***
